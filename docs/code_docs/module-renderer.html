Module: renderer

This file contains the main logic of the calculator such as parsing the inputed equation and using object of the MathEngine class to calculate the result, keyboard bindings, toggles between menus, repositioning the cursor in the input field, inserting results to the history and clearing the history and input fields. This file is binded to index.html.

(inner) calculate() → {void}

Attempts to calculate the result of the equation in the input field and show the outcome.

Attempts to calculate the result of the equation in the input field. If the calculation is successful, the result is displayed in the input field and the equation (and result) are added to the history. If the calculation fails, an error message is displayed in the history. History is scrolled to the right-bottom. All previous error messages are removed from the history.

(inner) insertToDisplayAtCursor(contentToInsert) → {void}

Inserts content to the input field at the cursor position. Also updates the cursor position.

Inserts the specified content to the input field at the cursor position. The cursor position is then updated to reflect the length and content of the inserted string. If the inserted content contains the character 'x', the cursor is positioned at the first occurence of 'x' in the inserted content (and 'x' is removed). In case of '^' character, the cursor is positioned depending on the character before the cursor in the input field. If the character is a number, right bracket or left bracket, the cursor is positioned at the end of the inserted content. Otherwise, the cursor is positioned in the place of x in the string
